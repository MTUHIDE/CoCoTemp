Getting Started

Step 1: Create Account

First, you will need to create an account. To do this click Register and enter your email, a username, and password.

Once you have created your account you will need to verify your email address by clicking the link in the confirmation email you were sent.

This link should then bring you back to the site and log you into your account.

Step 2: Construct iButton Sensor System

Now that you have made your account with CoCoTemp you will need to construct your iButton sensor.

In order to do this, you will need to get an iButton, a reader, a shield, and optionally an iButton keyfob.

The shield is needed to protect the iButton from the weather itself. At the same time making sure that the button is consistent temperature by protecting it from the sun, but also giving it access to the air.

You have multiple options for the shield. You can purchase your shield online or you could create your own shield at home. If you opt to build your own you can refer to our tutorial video below on how to do so, you will just need your iButton, a disposable cup(styrofoam or plastic), some string, an adhesive, and either your reader or keyfob to hold the iButton.

Step 3: Creating Sites and Adding Metadata

In order to create and manage sites make sure that you are signed in to your account. When you are ready to create a site go to My Sites.

Next, you will want to click on Create a new site.



From here you will need to give your site a name along with its location. You can set your site's location by clicking on the interactive map, or by entering the latitude, longitude, and elevation.

The next step is to give it a quick description. This way you will be able to tell which site it is in case you have many sites.

The final section of information to be entered is the globe site survey which asks for information about the location such as the height of the sensor, any obstacles, and other valuable information. You are not required to put all of this information in, but if you do then it will help others when comparing weather data.



Now that you have created a site you will be able to assign devices to it. This is done by clicking Add a device.

This will take you to a new page where you will be able to select a site and type of device via a drop-down menu. You will also want to put in the manufacturer/serial number so that you can tell the devices apart.

Note: the default device type is iButton so if you have a different type of temperature recording device make sure to change this.


Once you have created a site and a device you can select them from My Sites to view information about them such as the metadata and what devices are assigned to the site. When viewing a site it will also display a graph above of all of its current data that has been uploaded. On the side of the page, it will display the site's location on the map. In order to delete a device all you need to do is select the device and select delete.


Step 4: iButton and CoCo iTemp Application Usage

If you plan to use the CoCo iTemp desktop application to extract and/or upload data from your iButton you will need to install it. This can be downloaded by clicking here. In order to run the application you must have java installed, this can be downloaded here here. You will also need to download the appropriate one-wire drivers so the application can read data from the iButton these can be downloaded here.

To obtain iButton data you will need to plug it into your computers USB port via the reader, this must be done before starting the application or it will not detect the iButton. Next, start up the CoCo iTemp application and log into your account. Select the iButton from the dashboard. The iButton's associated site will appear, if the iButton does not have a site associated with it you can click the Assign Site to associate it with one of your sites. Once selected you can have your iButton start collecting data by clicking on the 'Mission' button. Next, you can start a new mission by clicking on the 'New Mission' button. You can stop a mission by clicking on 'Stop Mission'. On the mission page, you can also click on 'Read Data' to download the data from your iButton to a local file, this data can then be manually uploaded to the site if you prefer to do it this way.


To upload data make sure that you have your iButton selected and ensure that it has a site assigned to it. You can then click on the 'Upload' button and it will upload your data to the desired site. Another way to upload data to a site is to go to the Upload Data page. Here you can select the file that was generated by clicking on 'Read Data' from the iButton and the specific site that you would like to upload it to. You can also add notes to your upload on this page.



If you are using a different kind of software to obtain your data then make sure that the file you are using to upload is formatted as shown below. A good way to easily make sure of this is to open up your file in Microsoft Excel or a similar software and make sure that it is formatted as shown below. If you have a file that is not a CSV file that you wish to use make sure that you create an Excel file like the example and then you can save the file as a CSV by going to 'save file as' and then changing the file extension type to .csv rather than .xlsx.



Note: Currently an internet connection is needed in order to use the CoCo iTemp Application.


Step 5: Comparing Data using NOAA Sites

If you want to compare different site data you will want to go to the Microclimate Data page. Once there you can load NOAA sites based on custom filters using the filters tab on the right side of the screen.



You can also find CoCo Temp sites using the Microclimate Comparison Tool.



Once you have all of the desired sites loaded you can click on them and then click on 'Add to Graph'. If you want to remove something from the graph then re-click them on the map and click 'Remove from Graph'. Once you select a site the data will appear on the graph below and also in tables in the filter tab on the right.



When using the graphs some helpful tips can include being able to temporarily hide the data by clicking on the data's name in the legend on the right side. Note: The yellow lines on the graph are there to indicate large drops or rises in temperature which could help locate any suspect data.
